直方图均衡化

  • 图像处理

如果一副图像的像素占有很多的灰度级而且分布均匀，那么这样的图像往往有高对比度和多变的灰度色调。直方图均衡化就是一种能仅靠输入图像直方图信息自动达到这种效果的变换函数。它的基本思想是对图像中像素个数多的灰度级进行展宽，而对图像中像素个数少的灰度进行压缩，从而扩展像原取值的动态范围，提高了对比度和灰度色调的变化，使图像更加清晰。

直方图均衡化处理的“中心思想”是把原始图像的灰度直方图从比较集中的某个灰度区间变成在全部灰度范围内的均匀分布。直方图均衡化就是对图像重新分配图像像素值，也就是把给定图像的直方图分布改变成“均匀”分布直方图分布。
这里的均匀并不是直方图看起来高度一样，而是在一定灰度范围内，像素数是大致相等的。

设原始图像在(x，y)处的灰度为A，而改变后的图像为B，则对图像增强的方法可表述为将在(x，y)处的灰度A映射为B。在灰度直方图均衡化处理中对图像的映射函数可定义为:B = f(A)，这个映射函数f(A)必须满足两个条件:

  • F(x)在0≤xf≤L-1(其中L为图像的灰度级数)范围内是一个单值单增函数。这是为了保证增强处理没有打乱原始图像的灰度排列次序，原图各灰度级在变换后仍保持从黑到白(或从白到黑)的排列。
  • 对于0≤x≤L-1有0≤x≤L-1，这个条件保证了变换前后灰度值动态范围的一致性。
    原则上，只要满足上边两个条件，并尽可能将原来的灰度平均映射到0~L-1灰度阶的函数，都可以用来进行直方图均衡化处理。

累积分布函数(cumulative distribution function，CDF)即可以满足上述两个条件。

因为累积分布函数是单调增函数（控制大小关系），并且值域是0到1（控制越界问题），使用累积分布函数，像素映射可以保证原来的大小关系不变，较亮的区域，依旧是较亮的，较暗依旧暗，只是对比度增大，绝对不能明暗颠倒，并且如果是八位图像，那么像素映射函数的值域还是在0和255之间的，不会越界

我们可以直观看到上述公式是如何运作的：

实际处理步骤：

  1. 是统计直方图每个灰度级出现的次数
  2. 第二步是累计归一化的直方图
  3. 第三步是计算新的像素值

以下给出简要代码表示

第一步：

for(i=0;i<height;i++){
   for(j=0;j<width;j++){
      n[s[i][j]]++;
   }
}

for(i=0;i<L;i++){

    p[i]=n[i]/(width*height);

}

这里，n[i]表示的是灰度级为i的像素的个数，L表示的是最大灰度级，width和height分别表示的是原始图像的宽度和高度，所以，p[i]表示的就是灰度级为i的像素在整幅图像中出现的概率(其实就是p[]这个数组存储的就是这幅图像的归一化之后的直方图)。
第二步：

for(i=0;i<=L;i++){
   for(j=0;j<=i;j++){
      c[i]+=p[j];
   }
}

c[]这个数组存储的就是累计的归一化直方图。c[0]=0，c[L]=1。
第三步：

for(i=0;i<height;i++){
   for(j=0;j<width;j++){
      t[i][j]=c[s[i][j]]*L;
  }
}

t[][]就是最终直方图均衡化之后的结果。

对于彩色的图片来说，直方图均衡化一般不能直接对R、G、B三个分量分别进行上述的操作，而要将RGB转换成HSV来对V分量进行直方图均衡化的操作。

相关操作还有直方图匹配，对比直方图等。
